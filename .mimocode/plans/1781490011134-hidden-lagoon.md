# 压缩目录为ZIP文件

## 任务
将当前目录 `/home/xiang/Documents/agnet_1` 压缩为ZIP文件，不删除源文件。

## 实现方案

运行以下命令：
```bash
cd /home/xiang && zip -r agnet_1.zip agnet_1
```

- `-r` 参数：递归压缩所有子目录和文件
- 源文件保留（zip命令不会删除源文件）
- 生成的zip文件位于 `/home/xiang/agnet_1.zip`

## 验证
- 检查zip文件是否生成：`ls -lh /home/xiang/agnet_1.zip`
- 检查源目录是否完整：`ls /home/xiang/Documents/agnet_1/`
